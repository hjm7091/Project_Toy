# 웹 채팅 서비스(2020/06 ~ 2020/07)

### 1. 주제
##### Spring WebSocket을 이용한 간단한 채팅 서비스

### 2. 개발 범위
##### 채팅방 목록, 채팅방에 입장한 유저수 카운트, 채팅 기능

### 3. 개발 환경
* 프론트 웹 : freemarker, vue.js, axios.js, bootstrap 4.3.1
* 소켓 통신 라이브러리 : sockjs, stomp
* 서버 : springboot 2.1.15
* DB : redis
* 빌드 도구 : gradle-5.6.4
* 개발 도구 : sts-4.7.0
